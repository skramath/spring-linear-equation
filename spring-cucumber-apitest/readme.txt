Getting Started
	Api to check 2 variable linear equation. It include simple validation of the equation and also solving the equations. 
	Solved equation will return back the value of x and y

Features Include:*
Java 1.8
Spring boot 1.3.5.RELEASE
Maven 1.8
Xstream 1.4.9 message Convertors

Steps to Test :
1. mvn clean install
2. mvn test
